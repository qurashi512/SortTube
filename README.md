# 🚀 SortTube - Smart YouTube Organizer

**SortTube** is a professional Android application designed to help users manage their YouTube subscriptions efficiently. Instead of a messy subscription feed, users can categorize their favorite channels and playlists into beautifully designed custom folders.

---

## ✨ Key Features
* **Custom Folders:** Create, reorder, and manage folders with unique emojis and colors.
* **Advanced Organization:** Move channels and playlists into specific categories for easier access.
* **Modern UI:** Built with **Jetpack Compose** featuring a high-end Glassmorphism design and smooth animations.
* **Multi-Language Support:** Fully localized in **Arabic** and **English** with real-time switching.
* **Dark/Light Mode:** Intelligent theme switching for a comfortable viewing experience.

---

## 🛠 Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Material 3)
* **Architecture:** MVVM (Clean Architecture principles)
* **Dependency Injection:** Hilt (Dagger)
* **Database:** Room for local storage
* **Networking:** Retrofit & YouTube Data API v3
* **Concurrency:** Kotlin Coroutines & Flow
